#### 과제를 한 후
1. 새로운 방법은 딕셔너리를 통해 인덱스로 꺼내는 방식이 아닌 닉네임을 사용하듯 가지고 와서 편하게값을 찾는 방법을 다시 알게 되었다.
2. dictionary comprehension을 이용하여 반복문을 이용해 여러가지 방법으로 문제를 풀었는데 과제 내용에 나와있다시피 더 파이썬 다운 방식으로 문제 해결이 가능하다!!
3. 1학기때와 2학기 때 배운 걸 합쳐서 더 복잡하고 파이썬 다운 방식으로 문제 해결을 해볼 수 있는 좋은 계기였으며 더 많은 과제 문항이 있었으면 좋겠다는 생각이 들었다.
4. 그리고 for안에 추출한 키와 값들은 복사된 것이라 for밖에서 출력시 문제가 생기는 경우가 있어서 주의 해야된다.
